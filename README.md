# AimForge

Полноэкранный браузерный aim trainer с одной картой `Grind Shot 3x3`, меню по `Esc`, ручным вводом DPI и конвертацией сенсы между шутерами через реальные коэффициенты поворота. За основу взят профиль `CS2`, а остальные игры пересчитываются относительно него.

## Что внутри

- Полноэкранный 3D-тренажер с захватом мыши
- Главное меню:
  - `Тренироваться`
  - `Ввести текущие настройки`
  - `Настроить прицел`
  - `О разработчике`
- Поддержка игр:
  - `Counter-Strike 2`
  - `Valorant`
  - `Warzone`
  - `Apex Legends`
  - `PUBG`
  - `Deadlock`
  - `Rust`
  - `Fortnite`
  - `Rainbow Six Siege`
  - `Overwatch 2`
  - `The Finals`
  - `Escape from Tarkov`
- Подсказки после каждого раунда:
  - слишком высокая сенса
  - слишком низкая сенса
  - ранний выстрел
  - лишняя микро-коррекция
- Сохранение истории раундов в `localStorage`

## Локальный запуск

### macOS

```bash
./run-local.command
```

### Windows

```bat
run-local.cmd
```

### Через пакетный менеджер

```bash
pnpm install
pnpm dev
```

## Управление

- При открытии сайта показывается меню
- `Esc` открывает и закрывает меню
- Клик в сцену захватывает мышь
- ЛКМ стреляет по активной цели
- После завершения раунда меню открывается снова и показывает рекомендации

## Ограничения

- Браузер не разрешает включать fullscreen без действия пользователя, поэтому полноэкранный режим запускается по кнопке `Тренироваться`
- Физический DPI устройства браузер не отдает, поэтому значение вводится вручную
